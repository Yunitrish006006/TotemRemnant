# TotemRemnant

TotemRemnant is the optional death-backpack module for Totem.  It depends on
TotemCore, but not TotemNexus: a Nexus death node is supplied only through the
versioned optional lifecycle adapter when Nexus is installed.

The first extraction retains DeadRecall's addon API compatibility.  Production
code remains in DeadRecall until standalone, compatibility-bundle, legacy-world,
restart, multiplayer and Dedicated Server validation have all passed.
