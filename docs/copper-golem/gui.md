# 銅魁儡管理 GUI

使用銅板手 Shift+右鍵銅魁儡可開啟管理 GUI。每隻銅魁儡有自己的設定資料，銅板手只保存目前選取目標。

## 主要區塊

| 區塊 | 用途 |
| --- | --- |
| 模式 | 在分類與採集之間切換 |
| 運作／停止 | 控制是否開始工作 |
| 燃料槽 | 放入工作燃料 |
| 來源銅箱 | 顯示目前綁定的來源／Home |
| 分類目的地 | 管理容器、Prompt 與快取 |
| 採集工具欄 | 放入一件採集工具 |
| 採集倉庫 | 暫存最多 16 個採集物 |
| LLM | 設定 API URL、API Key、Model 並測試連線 |

## 欄位操作

| 欄位 | 左鍵 | 右鍵 |
| --- | --- | --- |
| 燃料槽 | 從主手或背包自動放入燃料 | 取回燃料 |
| 採集工具欄 | 從主手或背包自動放入工具 | 取回工具 |
| 採集倉庫 | 無 | 取回採集物 |

## 模式切換限制

切換模式前必須先停止銅魁儡。下列任一條件成立時，伺服器會拒絕切換：

- 銅魁儡手上仍有分類貨物。
- 採集工具仍留在工具欄。
- 採集倉庫仍有物品。

伺服器是設定與欄位資料的最終權威；client GUI 只負責顯示與送出操作請求。
