# 銅魁儡管理 GUI

使用銅板手 Shift+右鍵銅魁儡可開啟管理 GUI。每隻銅魁儡有自己的設定資料，銅板手只保存目前選取目標。

## 主要區塊

| 區塊 | 用途 |
| --- | --- |
| 模式 | 在分類與採集之間切換 |
| 運作／停止 | 控制是否開始工作 |
| 燃料槽 | 透過原版 slot 放入工作燃料 |
| 來源銅箱 | 顯示目前綁定的來源／Home |
| 分類目的地 | 管理容器、Prompt 與快取 |
| 採集工具欄 | 透過原版 slot 放入一件採集工具 |
| 採集倉庫 | 暫存最多 16 個採集物，可用原版 slot 取回 |
| 玩家背包 | 右半邊顯示原版風格背包與快捷欄 |
| LLM | 設定 API URL、API Key、Model 並測試連線 |

## 欄位操作

銅魁儡管理 GUI 是容器型介面。燃料、採集工具、採集倉庫與玩家背包都走原版 slot transaction，不再使用舊的主手自動放入按鈕。

| 欄位 | 支援操作 | 限制 |
| --- | --- | --- |
| 燃料槽 | 拖曳放入、取出，或從玩家背包 Shift-click 移入 | 只接受合法燃料；已有燃料時需相同 Item 與 Components |
| 採集工具欄 | 拖曳放入、取出，或從玩家背包 Shift-click 移入 | 只接受合法採集工具；最多一件 |
| 採集倉庫 | 拖曳取出或 Shift-click 取回 | 不允許玩家手動放入物品；最多 16 個採集物 |
| 玩家背包 | 原版拖曳、交換、拆半、合併與 Shift-click | 銅魁儡 slot 由伺服器驗證 |

運作中調整燃料、工具或採集倉庫可能被伺服器拒絕；需要更換工具或清空倉庫時，先停止銅魁儡。

## 模式切換限制

切換模式前必須先停止銅魁儡。下列任一條件成立時，伺服器會拒絕切換：

- 銅魁儡手上仍有分類貨物。
- 採集工具仍留在工具欄。
- 採集倉庫仍有物品。

伺服器是設定與欄位資料的最終權威；client GUI 只負責顯示與送出操作請求。
