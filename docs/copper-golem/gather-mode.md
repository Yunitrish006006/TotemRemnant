# 銅魁儡採集模式

採集模式會讓銅魁儡在指定工作區內尋找可採集方塊，使用工具破壞後，把掉落物暫存在採集倉庫，再帶回來源銅箱。

## 設定流程

1. 用銅板手 Shift+右鍵銅魁儡。
2. 停止銅魁儡並切換到「採集」。
3. 右鍵銅箱，設定 Home／來源銅箱。
4. 右鍵普通方塊設定 Corner A。
5. Shift+右鍵普通方塊設定 Corner B。
6. 左鍵普通非容器方塊，新增或移除該 Block ID 的手動採集目標。
7. 在 GUI 放入工具與燃料。
8. 切換為「運作」。

若採集 LLM 已啟用且設定完整，手動採集目標可以為空。

## 工作區與掃描

- 兩個角點會正規化成完整工作區。
- 每軸最多 64 格，總體積最多 262144 格。
- 只掃描已載入區塊，不會強制載入區塊。
- 從工作區最上層往下搜尋。
- 每 tick 最多檢查 512 個候選方塊。
- 成功採集後保存掃描游標，下次從上一位置後方繼續。

## 採集限制

- 排除來源銅箱、已綁定容器、其他容器、流體與不可破壞方塊。
- 必須有合法且適用的工具。
- 採集倉庫最多保存 16 個物品。
- 同一趟只接受相同 Item 與 Data Components 的掉落物。
- 成功破壞方塊後才消耗燃料與工具耐久。
- 路徑失敗或長時間無法接近的方塊會暫時跳過。

倉庫滿 16 個，或完整掃描後找不到可繼續合併的目標時，銅魁儡會返回 Home 存放物品。
