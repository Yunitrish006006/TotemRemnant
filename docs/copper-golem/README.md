# 銅魁儡使用指南

DeadRecall 的銅板手可選取並管理銅魁儡。每隻銅魁儡保存自己的來源銅箱、模式、燃料、目的地、採集區域與 LLM 設定。

## 模式概覽

| 模式 | 用途 | 來源銅箱 |
| --- | --- | --- |
| 分類 | 從來源銅箱取物並分配到綁定容器 | 待分類物品來源 |
| 採集 | 在工作區採集並把物品帶回 | Home／存放倉庫 |

分類與採集共用同一個來源銅箱，但用途不同。每隻銅魁儡只能設定一個來源銅箱。

## 快速開始

### 選取銅魁儡

手持銅板手，Shift+右鍵銅魁儡。板手會綁定該銅魁儡並開啟管理 GUI。

### 設定來源

手持已綁定的銅板手右鍵銅箱，將其設定為來源銅箱。左鍵目前來源銅箱可解除綁定。

### 分類

1. 在 GUI 停止銅魁儡並切換到分類模式。
2. 右鍵一般容器加入目的地。
3. 放入燃料。
4. 切換為運作。

### 採集

1. 在 GUI 停止銅魁儡並切換到採集模式。
2. 右鍵方塊設定 Corner A，Shift+右鍵設定 Corner B。
3. 左鍵方塊新增手動採集目標。
4. 放入工具與燃料。
5. 切換為運作。

## 基本操作

| 操作 | 效果 |
| --- | --- |
| 銅錠右鍵受傷銅魁儡 | 消耗一個銅錠並修復 4 點生命值；創造模式不消耗 |
| 銅板手 Shift+右鍵銅魁儡 | 選取並開啟 GUI |
| 右鍵銅箱 | 設定來源銅箱 |
| 左鍵來源銅箱 | 解除來源 |
| 分類模式右鍵一般容器 | 加入分類目的地 |
| 分類模式左鍵已綁定容器 | 移除目的地 |
| 採集模式右鍵方塊 | 設定 Corner A |
| 採集模式 Shift+右鍵方塊 | 設定 Corner B |
| 採集模式左鍵普通方塊 | 切換手動採集目標 |

跨維度綁定不被允許。銅魁儡、來源、目的地與工作區必須位於同一維度。
