# 銅魁儡分類模式

分類模式會讓銅魁儡從唯一的來源銅箱取出物品，再依綁定順序放入一般容器或容器內的 DeadRecall 背包。

## 設定流程

1. 手持銅板手，Shift+右鍵銅魁儡。
2. 在 GUI 中停止銅魁儡並切換到「分類」。
3. 右鍵銅箱，設定唯一來源銅箱。
4. 右鍵一般容器，加入分類目的地。
5. 在 GUI 放入燃料。
6. 切換為「運作」。

左鍵來源銅箱可解除來源；左鍵已綁定的一般容器可移除目的地。

## 分類規則

- 每次最多從來源銅箱取出 16 個物品。
- 來源銅箱只作為來源，不可成為分類目的地。
- 目的地按綁定順序檢查，不直接使用最近距離決定。
- 容器內已有相同 Item 與 Data Components 時，可合併或使用空格。
- 容器內的普通 DeadRecall 背包也可以作為分類目標。
- 死亡背包不會被當作分類背包。
- 單純空箱不會自動吃掉所有物品。
- 所有目的地都無法接收時，銅魁儡會嘗試把物品放回來源銅箱。

## LLM 與快取判斷順序

1. 拒絕快取命中時直接拒絕。
2. 已有相同 Item + Components 且可合併時直接接受。
3. 接受快取命中且容器有空位時接受。
4. 目的地啟用 LLM、Prompt 完整且尚未快取時，送出判斷請求，本輪先跳過。
5. LLM 結果會寫入該容器的接受或拒絕快取。

LLM 設定方式見 [LLM 設定](llm.md)。
