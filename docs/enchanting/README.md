# 附魔台與雕紋書櫃

DeadRecall 修改附魔台的書櫃力量計算。附魔台會讀取原版有效書櫃位置上的**雕紋書櫃**，並依書櫃內實際放入的書本計算附魔力，而不是把每個書櫃固定視為一點力量。

## 附魔力計算

| 書本內容 | 附魔力 |
| --- | ---: |
| 普通書 | 1 |
| 附魔書 | 書上所有附魔等級的總和 |
| 空槽或其他物品 | 0 |

例如：

- 一個雕紋書櫃放入 6 本普通書，提供 6 點附魔力。
- 一本同時帶有等級 3 與等級 2 附魔的附魔書，提供 5 點附魔力。
- 多個雕紋書櫃的有效書本力量會累加。

附魔台使用的總附魔力上限為 **64**。

## 附魔選項

DeadRecall 取代原版 15 書櫃上限，使用最高 64 點附魔力重新計算三個附魔選項。

- 第一格仍是較低成本選項。
- 第二格仍是中等成本選項。
- 第三格會使用較高附魔力，最高成本限制為 64。
- 物品必須具有可附魔資料，否則不會產生附魔選項。

## 擺放方式

雕紋書櫃必須放在附魔台原版會檢查的書櫃位置。附魔台與書櫃之間應保留空氣，確保粒子可以正常顯示。

## 粒子效果

雕紋書櫃會向附魔台產生附魔粒子：

- 普通書越多，粒子出現頻率越高。
- 附魔書的附魔等級越高，粒子頻率、速度與單次數量越高。
- 粒子只提供視覺回饋，不會額外改變附魔力。

## 目前限制

- 附魔力來源是雕紋書櫃內的書本。
- 總附魔力超過 64 的部分不會提高附魔選項。
- 粒子強度另外有視覺上限，避免大量書本造成過多粒子。
