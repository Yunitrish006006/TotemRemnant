# 背包系統

DeadRecall 提供四級一般背包，以及獨立的死亡背包與整理功能。

## 文件

- [背包等級與防護](tiers.md)
- [死亡背包與 `/back`](death-backpack.md)
- [整理功能](sorting.md)

## 快速說明

一般背包右鍵開啟，內容使用原版 Container Data Component 儲存。背包不可堆疊，並支援 Shift+點擊快速移動。

死亡背包會在原版生成世界掉落物之前，直接從玩家的 Server 權威物品欄與裝備資料封裝一般物品；DeadRecall 背包本身維持獨立掉落。死亡背包具備永久保存、紅色光柱與虛空保護。
