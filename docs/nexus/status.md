# Totem Nexus 實作狀態

此頁以 Space Unit 初版 commit `ab22c5c5a65ed9299760e694baaa44d9460df48c` 為基礎，並包含後續的定向結構刷新修正。

## 已完成基礎

- Space Unit 世界 SavedData。
- 玩家探索資料 SavedData。
- 磁石 Space Unit 註冊與羅盤綁定。
- 羅盤左鍵探索磁石。
- 私有 Owner 權限基礎。
- 探索與權限雙重過濾。
- 以磁石或玩家位置作為來源的地圖請求。
- 相對位置 Space Unit 地圖 GUI。
- Dimension 分頁、清單、縮放、節點選取與重新整理。
- Server-side 傳送報價。
- 傳送 session、倒數、重新驗證、成本扣除與安全落點。
- 死亡背包成功建立後，自動建立死亡 Space Unit 節點基礎。
- 中文繁體、中文簡體與英文翻譯鍵。
- 開啟磁石來源地圖時重新掃描來源結構。
- 選取地圖目標時，只重新掃描來源與選取目標並刷新報價。
- 啟動傳送前再次重新掃描來源與目標，避免使用過期結構快照。

## 已有資料但效果尚未完整套用

- 傳送最大水平偏差報價。
- 抵達傷害機率報價。
- 石碑結構快照、tier、resonance 與穩定度資料。

目前 GUI 可以顯示這些報價，但實際隨機偏差、抵達傷害與石碑磨損效果仍未完整實作。

## 尚未完成

- 磁石註冊確認流程。
- 節點改名、可見性與管理 GUI。
- 地圖搜尋、分類與好友篩選。
- 傳送偏差隨機化。
- 抵達傷害實際效果。
- 石碑磨損、退化與手動重新校準介面。
- 好友邀請、好友磁石與人體磁石。
- 分散重生 Gamerule 與密度加權分配。
- 死亡節點回收、隱藏與完整成本整合。
- Totem 模組拆分與舊 identifier migration。
