# 煉金配方

煉藥鍋配方資料位於：

```text
src/main/resources/data/deadrecall/deadrecall/cauldron_recipes/
```

目前文件化的配方：

| 檔案 | 結果 | 核心材料 |
| --- | --- | --- |
| `saltpeter.json` | 硝石 | 木灰、蘑菇、豬糞 |
| `hot_cocoa.json` | 熱可可 | 牛奶、裝可可粉的缽、糖 |
| `cherry_brew.json` | 櫻花釀 | 糖、櫻花樹葉、發光莓、甜莓 |

另有一般工作台或熔爐配方，例如木灰、缽與煉金火藥。精確材料數量、投料順序、加熱條件與等待時間應以 JSON 資料檔為準。
