# 煉金系統

DeadRecall 的生存煉金內容包含豬糞、木灰、硝石、缽與資料驅動煉藥鍋配方。

## 文件

- [材料](materials.md)
- [配方](recipes.md)

## 快速說明

- 豬糞可由豬吃草後產生的帶豬糞地面方塊取得。
- 木灰由乾草捆熔煉取得。
- 硝石透過營火加熱的滿水煉藥鍋煉製。
- 缽可承裝材料，也可對硫磺方塊互動取得帶硫磺的缽。
